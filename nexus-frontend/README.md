# ADP Nexus HCM Platform - Frontend

This is the Angular frontend for the **ADP Nexus HCM Platform**. 

Please see the [Main Project README](../README.md) located in the parent directory for complete instructions on how to start the XAMPP Database, Spring Boot Backend, and this Angular Frontend.

### Quick Start
If you already have the backend running:
```bash
ng serve -o
```
Navigate to `http://localhost:4200/`.
